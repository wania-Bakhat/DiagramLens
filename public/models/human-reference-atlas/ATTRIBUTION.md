# Human Reference Atlas 3D assets

DiagramLens bundles these HuBMAP Human Reference Atlas files as reference anatomy assets:

| Local file | Source file | Source release |
| --- | --- | --- |
| `heart.glb` | `VH_M_Heart.glb` | HuBMAP CCF 3D Reference Object Library v1.2 |
| `brain.glb` | `Allen_F_Brain.glb` | HuBMAP CCF 3D Reference Object Library v1.2 |
| `lungs.glb` | `VH_M_Lung.glb` | HuBMAP CCF 3D Reference Object Library v1.2 |
| `kidney-left.glb` | `VH_M_Kidney_L.glb` | HuBMAP CCF 3D Reference Object Library v1.2 |
| `kidney-right.glb` | `VH_M_Kidney_R.glb` | HuBMAP CCF 3D Reference Object Library v1.2 |
| `eye-left.glb` | `VH_M_Eye_L.glb` | HuBMAP CCF 3D Reference Object Library v1.2 |

The CCF 3D Reference Object Library is released under the [Creative Commons Attribution 4.0 International license](https://creativecommons.org/licenses/by/4.0/). The library describes its organs as medically illustrated reference objects reviewed by organ experts. The brain is based on the Allen Human Reference Atlas – 3D, 2020.

Attribution: Browne, K., Schlehlein, H., Herr II, B. W., Quardokus, E., Bueckle, A., & Börner, K. (2022). *HuBMAP CCF 3D Reference Object Library* (v1.2).

Sources:

- https://hubmapconsortium.github.io/ccf/pages/ccf-3d-reference-library.html
- https://github.com/hubmapconsortium/ccf-releases/tree/main/v1.2/models
- https://apps.humanatlas.io/kg-explorer/?do=ref-organ

The liver and pancreas are streamed directly from the corresponding HuBMAP
Human Reference Atlas digital-object records at runtime. They retain the same
CC BY 4.0 attribution and are not redistributed with this project.
